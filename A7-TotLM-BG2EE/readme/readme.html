Overview

This mod makes the Icewind Dale expansion "Trials of the Luremaster" available to BG2:EE (2.0 or higher) and EET (Enhanced Edition Trilogy).

The quest starts by talking to the halfling Hobart Stubbletoes, who can be found in the Umar Hills Inn. In the course of the adventure you have the opportunity to explore a haunted castle, test your mettle in challenging combat and lift an ancient curse.

Visit the SHS Forums or the Beamdog Forums if you have questions, or download the latest version from Spellhold Studios or GitHub.

Installation

This is a WeiDU mod, that means it is very easy to install. Simply unpack the downloaded archive into your game's installation directory (where you can find the file "chitin.key") and run "setup-A7-TotLM-BG2EE.exe" (Windows) or "setup-A7-TotLM-BG2EE.command" (macOS). Follow the instructions and you are ready to start.

It is not required to start a new game after installing the mod. However, you must install it before entering the Umar Hills Inn for the first time.

To uninstall, run "setup-A7-TotLM-BG2EE.exe" or "setup-A7-TotLM-BG2EE.command" again and follow the prompts.

Compatibility

The mod is available for BG2:EE 2.0 or higher and EET.

Sword Coast Stratagems (SCS) v32 component "Smarter Beholders" interferes with the beholder scripts of this mod.

Mod Components

  1. Trials of the Luremaster for BG2:EE (requires BG2:EE or EET patched to 2.0 or higher)
    This is the main component. It installs many new areas, creatures, items, spells and everything else needed to enjoy the adventure.

    The mod makes a number of slight changes to the adventure to bring it more in line with BG2. Spells and items that are only available in Icewind Dale and have little or no relevance in the adventure have been replaced by similar BG2 spells or items. Moreover, a small number of creature animations and scripts have been adapted or slightly improved.

  2. Make IWD spells available to the player (requires the main component)
    The mod installs a number of spells unique to Icewind Dale, which are used by the various creatures in the course of the adventure. This component makes these spells available to the player and adds them to a number of stores, creatures and containers.
    Note: It is not recommended to install this component together with IWDification or the IWD spells components of Sword Coast Stratagems (SCS).

  3. Rebalance weapons and equipment (requires the main component)
    Many items in Icewind Dale are exceptionally powerful. Since the mod allows you to retrieve a good number of powerful weapons, armor and equipment, it may diminish the challenge from the rest of the game. This component replaces these items by less powerful variants.
